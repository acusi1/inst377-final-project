About This Project

This web application is a full-stack Customer Management System built using Node.js, Express, and Supabase. It allows users to add and view customer information stored in a cloud database.

Key Features

  • Add new customers with first name, last name, and state
  • View all stored customers from the database
  • Fetch random facts from an external API
  • Real-time updates using Fetch API

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js + Express
  • Database: Supabase
  • External API: Useless Facts API

Purpose

The goal of this project is to demonstrate full-stack development skills including API design, database integration, and frontend-backend communication.
